# 📚 GDZ Bot — Homework Solutions Telegram Bot

Telegram-бот для поиска решений домашних заданий с сайта gdz.ru. Пользователь пишет запрос в свободной форме — бот сам находит нужный учебник через DuckDuckGo, парсит страницу через Playwright и возвращает скриншот с решением.

## Возможности

- **Свободный запрос** — просто напиши `математика 5 класс 123 упр`, не нужно знать URL
- **Автоматический поиск учебника** — DuckDuckGo ищет нужную книгу на gdz.ru без блокировок
- **Headless парсинг** — Playwright открывает страницу как настоящий браузер, обходя защиту
- **Скриншот решения** — возвращает изображение с ответом на упражнение
- **Выбор учебника** — если найдено несколько вариантов (разные авторы), предлагает выбрать
- **Пагинация** — листание списка учебников прямо в боте
- **Grid изображений** — несколько изображений ответа склеиваются в одну картинку

## Стек

| Компонент | Технология |
|---|---|
| Язык | Python 3.11+ |
| Telegram | aiogram 3+ |
| Браузер | Playwright (Chromium headless) |
| Обработка изображений | Pillow |

## Установка

```bash
git clone https://github.com/kurumi-mProject/gdz-bot.git
cd gdz-bot
python -m venv venv
source venv/bin/activate
pip install aiogram Pillow playwright
playwright install chromium
```

Вставь токен бота в `bot.py`:
```python
BOT_TOKEN = "your_token_here"
```

Запуск:
```bash
python bot.py
# или
bash run.sh
```

## Использование

Отправь боту запрос в формате:

```
математика 5 класс 123 упр
русский язык 7 класс 45 упр
физика 9 класс 12 упр
алгебра 8 класс 356 упр
```

Бот:
1. Ищет учебник на gdz.ru через DuckDuckGo
2. Если нашёл один вариант — сразу открывает упражнение
3. Если нашёл несколько — предлагает выбрать нужный
4. Делает скриншот страницы с решением
5. Отправляет изображение

## Как работает парсинг

```
Запрос пользователя
       ↓
Парсинг regex: предмет + класс + номер упражнения
       ↓
DuckDuckGo: "gdz.ru {предмет} {класс} класс"
       ↓
Playwright открывает найденный URL gdz.ru
       ↓
Поиск кнопки/ссылки нужного упражнения
       ↓
Скриншот блока с решением
       ↓
Отправка пользователю
```
